# 🚑 RoadSOS – Smart Emergency Response System

RoadSOS is a real-time emergency response platform designed to reduce response time during road accidents using AI routing, offline fallback, and live tracking.

---

## 🔥 Features

* 🚨 One-tap SOS activation
* 📍 Real-time location tracking
* 🏥 Smart hospital matching (AI-based)
* 🚓 Police & ambulance notification
* 📡 Offline mode with SMS fallback simulation
* 👤 Patient profile auto-sharing (blood group, allergies, contacts)
* 📊 Admin dashboard for monitoring incidents

---

## ⚙️ Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Node.js (in progress)
* Database: SQLite (local storage)
* Maps: Leaflet.js

---

## 🧠 How It Works

1. User presses SOS
2. Location + medical data captured
3. System finds nearest hospital
4. Emergency services notified
5. Live tracking enabled
6. Offline fallback triggers SMS protocol if network fails
